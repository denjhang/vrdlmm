ツールの説明

○wave2mpd.exe
　説明
　　mmldrv専用PCMデータ作成ツールです
　　wavファイルよりmpd形式のデータに変換するツールです。
　
　使い方
　　起動するとウィンドウが表示されるのでwavをドラッグします。
　　同じフォルダにmpd形式のファイルが作成されます。
　　mml上はxxxxx.wavと記載していたものをxxxxx.mpdに
　　変更するだけで利用できます。

○edit32sample
　説明
　　波形編集用ソフトです
　　mmldrvの@wコマンドで定義する波形を作成するためのものです。
　　SPPCMに対応しておりSCCI無しでプレビュー再生が可能です。

　使い方
　　起動するとウィンドウが表示されます。
　　右側が波形編集エリアで、マウスで波形を描きます。
　　左側はモード設定及びインフォメーション表示です。

　　・Edit Mode
　　　「8bit Wave Mode」「4bit Wave Mode」をクリックすると
　　　モードが選択され選択されているモードの背景が薄い紫になります。
　　　このモードは波形編集の精度を8bitで行うか4bitで行うかが変わります。
　　　※出力されるデータは8bitです。

　　・Synthesis Mode
　　　「WSG」「SCC」が選択できます。選択方法はEdit Modeと同様です。
　　　プレビュー用の再生で、WSGライクな波形合成を行うかSCCライクな波形合成を
　　　行うかを選択します。

　　・基本的なキー操作
　　　カーソルキーの上下　：プレビューのオクターブを変更させます。
　　　「CFVGBNJMK<L>?」 　：音階を再生します。
　　　CTRL+C　　　　　　　：表示されている波形データをカンマ区切りの
　　　　　　　　　　　　　　テキストデータとしてクリップボードへコピーします。
　　　CTRL+V　　　　　　　：クリップボード上のテキストを波形として貼り付けます。
　　　　　　　　　　　　　　※変換できないテキストの場合は貼り付けできません。
